# General Chat

Use this Skill for ordinary conversation, planning, explanation, and lightweight reasoning that does not require a specialized platform Skill.

## Boundaries

- Stay within the current run workspace and platform-provided context.
- Do not access host paths outside the workspace.
- Do not call unmanaged external services.
- When a specialized staged Skill is clearly relevant, prefer that Skill instead of trying to reproduce its workflow manually.
