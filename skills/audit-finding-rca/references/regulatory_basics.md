# 核心法规条款速查

Use this reference when a finding cites GMP, MAH, ICH, EU GMP, data integrity, aseptic, validation, or quality-system obligations.

## 中国 GMP（2010版）关键章节

| 章节 | 条款范围 | 常见审计关联 |
|------|---------|-------------|
| 第二章 质量管理 | 第5-17条 | 质量体系缺陷、偏差/CAPA/OOS管理不足 |
| 第三章 机构与人员 | 第18-28条 | 人员培训不充分、职责不清、人员不足 |
| 第四章 厂房与设施 | 第29-56条 | 洁净区管理、HVAC、压差/温湿度控制 |
| 第五章 设备 | 第57-90条 | 设备确认、校准、维护、计算机化系统 |
| 第六章 物料与产品 | 第91-130条 | 供应商管理、物料验收、储存条件、追溯 |
| 第七章 确认与验证 | 第138-150条 | 工艺验证、清洁验证、分析方法验证不足 |
| 第八章 文件管理 | 第151-192条 | SOP缺失/过期、记录不完整、数据完整性 |
| 第九章 生产管理 | 第193-227条 | 污染/交叉污染、清场、批记录、过程控制 |
| 第十章 质量控制 | 第217-247条 | 取样、OOS、稳定性、留样、数据完整性 |
| 第十一章 委托生产与委托检验 | 第278-296条 | 委托协议、质量责任划分、审计权利 |
| 第十二章 产品发运与召回 | 第297-309条 | 发运记录、召回程序 |
| 第十三章 自检 | 第310-312条 | 自检频次、范围、整改跟踪 |

## NMPA MAH 关键法规

### 132号文：《药品上市许可持有人委托生产监督管理规定》

| 条款 | 要点 |
|------|------|
| 第四条 | MAH对委托生产药品质量全面负责，建立全生命周期质量保证体系 |
| 第八条 | MAH对受托方进行现场审计，确认质量管理体系符合要求 |
| 第十条 | 质量协议必须明确双方职责、沟通机制、变更管理 |
| 第十四条 | MAH建立上市后风险管理计划，监督受托方偏差/CAPA/OOS处理 |
| 第十七条 | MAH对受托方进行定期审计，通常至少每年一次 |
| 第二十条 | 跨省委托需两地药监联合监管 |

### 134号文：《药品上市许可持有人药品质量主体责任及质量管理体系建设指南》

| 条款 | 要点 |
|------|------|
| 第二条 | MAH质量管理体系覆盖产品全生命周期 |
| 第六条 | MAH应建立独立 QA 团队，具备对受托方的监督能力 |
| 第八条 | 变更管理：MAH评估变更影响，受托方实施前需MAH批准 |
| 第十一条 | 投诉、不良反应、召回由 MAH 主导，受托方配合 |
| 第十三条 | MAH需设立药品安全委员会和质量受权人 |

## 国际法规

- ICH Q7 §2 Quality Management: 质量方针、质量职责、偏差/CAPA。
- ICH Q7 §3 Personnel: 人员资质、培训、卫生。
- ICH Q7 §5 Process Equipment: 设备设计、清洁、维护、校准。
- ICH Q7 §12 Validation: 工艺验证、清洁验证、分析方法验证。
- ICH Q7 §14 Materials Management: 物料追溯、供应商管理。
- ICH Q10 §1.5: PQS 要素，包括工艺性能和产品质量监控、CAPA、变更、管理回顾。
- ICH Q10 §3.2: CAPA 应基于根因分析和风险分级，并核查有效性。
- EU GMP Annex 1: 无菌制造、HVAC/洁净区分级、环境监测、人员行为。
- EU GMP Annex 2: 生物制品起始物料控制、病毒清除验证、生物负载监控。

## 速查

- 委托生产责任划分：优先 132号文第10/14/17条。
- MAH质量体系：优先 134号文第2/6/8条。
- 数据完整性：优先 GMP 第八章、ICH Q7 §6、计算机化系统权限/审计追踪要求。
- 无菌保证：优先 EU GMP Annex 1、GMP 第四章。
